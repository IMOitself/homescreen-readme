## Feats and Progress

| Features |
| --- |
✅ home screen widget
✅ update widget
✅ remember widget content
✅ run bash commands via [AfterRun](https://github.com/IMOitself/AfterRun/tree/966e6317eba59add8cb193e976b77d6ca439bd8c) and Termux

⬜ get README.md

⬜ commit locally
